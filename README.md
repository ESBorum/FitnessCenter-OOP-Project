# Fitness Center Booking System (OOP Project)

## Description
This project is a console-based fitness center booking system built using Object-Oriented Programming principles in Java.

## Features
- Create members
- Create activities (Strength & Cardio)
- Book and cancel activities
- Custom exception handling
- Generic repository (Repository<T>)
- Lambda sorting and filtering
- Multi-threading (booking confirmation)

## OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstract class (Activity)
- Interface (Bookable)

## Technologies
- Java
- Collections (ArrayList)
- Streams & Lambda
- Threads
